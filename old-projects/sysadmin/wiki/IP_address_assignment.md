IP address assignment
=====================

We are assigned the IPv6 address range: 2001:4f8:3:36:bl04::01 - 0F for
virtual servers.

huchra is on 2001:4f8:3:36:bl04::01

the default gw for ipv6 traffic is on 2001:4f8:3:36::1.
